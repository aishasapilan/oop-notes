Exception Handling
Different of exceptions in phyton
*SyntasError - misspelled keyword, missing colon or unbalanced parenthesis.
*TypeError - an object of the wrong type   ex:adding string to integer.
*NameError - variable or function name is not found
*IndexError - index out of range 
*KeyError - when key is not found in a dictionary
*ValueError - an invalid input or argument    ex:converting a string
*AttributeError - an attribute or method is not found on an object
*IOError - an I/O operation fails due to an input/output error
*ZeroDivissionError - dividing a number by zero 
*ImportError - import statement fails to find or load a module.
